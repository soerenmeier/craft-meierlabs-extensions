# Meier Labs Extensions for Craft CMS

## Overview

Meier Labs Extensions is a small plugin containing little tweaks and extensions for Craft CMS.

## Features

- Simpler Link field design
- Allow to ungroup Matrix field buttons
- Hide more ul and ol options in CKEditor
- Set cke font-size to 16px

## Installation

Via the [Craft Plugin Store](https://plugins.craftcms.com/meierlabs-extensions?craft5) or the command-line:

```bash
composer require soerenmeier/meierlabs-extensions -w && php craft plugin/install meierlabs-extensions
```
